# Electron桌面软件开发

## Electron 官方示例代码

```bash
$ git clone https://github.com/electron/electron-api-demos
$ cd electron-api-demos
$ npm install # 需要配置镜像 或者vpn
$ npm start
```

执行npm start 之后会展开一个官方示例的一个界面软件,大致展示electron部分功能
