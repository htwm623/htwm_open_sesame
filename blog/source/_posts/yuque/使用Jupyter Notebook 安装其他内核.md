## javascript 内核安装
参考[ijavascript](https://github.com/n-riesco/ijavascript)<br />

1. windows 
```bash
pip3 install --upgrade pip
pip3 install jupyter
npm install -g ijavascript
ijsinstall
```

2. 正常运行 jupyter notebook 选择node 的内核即可
