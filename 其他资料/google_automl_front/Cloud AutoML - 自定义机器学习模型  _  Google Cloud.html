训练自定义机器学习模型

Cloud AutoML 是一套机器学习产品，可让机器学习知识有限的开发者根据其业务需求训练出高质量的模型。 它以 Google 最先进的迁移学习技术和神经架构搜索技术为基础。

领先的性能

Cloud AutoML 运用 Google 研究部门十余年来积淀的专有技术，帮助您的机器学习模型实现更卓越的性能和更准确的预测。

快速上手使用

借助 Cloud AutoML 简单的图形界面，您可以基于自己的数据来训练、评估、改进和部署模型。您只需几分钟便可拥有自己的自定义机器学习模型。

生成高质量的训练数据

Google 的人工标签服务可安排一组人员为您标注或清理标签，确保您的模型能够使用高质量数据进行训练。

AutoML 工作原理

Cloud AutoML 特性

自定义模型

只需极少的工作量和机器学习专业知识，即可针对您的业务需求训练自定义机器学习模型。

由 Google 的 AutoML 和迁移学习技术驱动

利用 Google 先进的 AutoML 和迁移学习技术来创建高质量的模型。

完全集成

Cloud AutoML 可与其他 Google Cloud 服务完全集成，让您能够以统一的方式访问所有 Google Cloud 服务，包括将您的训练数据存储在 Cloud Storage 中。要根据训练后的模型生成预测，您只需在使用现有 Vision API 时添加一个参数来指定您的自定义模型，或使用 AI Platform 的在线预测服务。

与数据标签服务集成

如果您有图片但没有标签，可以请 Google 内部团队根据您的专门说明对图片进行分类。您不但可以获得高质量的训练数据，同时还可确保数据的私密性。您还可以无缝使用人工标注过的数据来训练自定义模型。仅适用于 AutoML Vision。
